🚀 NexGen – Modern Responsive Website
📌 Overview

NexGen is a fully responsive, modern business website designed to showcase digital services with a clean UI, smooth animations, and optimized performance.
It demonstrates strong fundamentals in HTML, CSS, and JavaScript, along with responsive design principles.

✨ Features
📱 Fully Responsive Design (Mobile, Tablet, Desktop)
🎨 Modern UI/UX Layout
🍔 Interactive Hamburger Menu
⚡ Smooth Scroll Animations (Intersection Observer)
🧩 Reusable Components (Cards, Buttons, Grid System)
📬 Newsletter Form Integration
🎯 Clean and Maintainable Code Structure
🛠️ Tech Stack
HTML5 – Semantic structure
CSS3 – Flexbox, Grid, Custom Properties
JavaScript (Vanilla) – DOM manipulation & interactions
Font Awesome – Icons
Google Fonts (Inter) – Typography

⚙️ Key Implementations
🔹 Responsive Navigation
Desktop: Horizontal menu
Mobile: Hamburger toggle menu
Smooth slide-in effect using CSS transitions
🔹 Animation System
Implemented using Intersection Observer API
Elements animate on scroll (.animate-from-left)
🔹 Layout System
CSS Grid + Flexbox hybrid approach
Scalable and reusable utility classes

🎯 Learning Outcomes
Practical implementation of responsive web design
DOM manipulation and event handling
Structuring scalable CSS architecture
Debugging layout and interaction issues
